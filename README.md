## 🚀 Projetos em Destaque

### 🏭 Batch Tracking — Rastreamento de Lote Industrial
> Algoritmo de rastreamento retroativo de lotes ao longo de ~15 áreas do processo fabril

- Lógica de varredura reversa utilizando **PI AF Event Frames** como flags de passagem de produto
- Consolidação de parâmetros de processo e qualidade por etapa
- Report **Power BI** integrado com API JSON do sistema de rastreabilidade de matéria-prima
- Cruzamento de notas fiscais, volumes de entrada e histórico completo de fabricação

**Stack:** `PI AF` `Event Frames` `PI Tags` `Power BI` `API JSON` `Python`

---

### 🚨 Plataforma de Alarmes Industriais
> Monitoramento de alarmes com foco em Segurança de Processos

- Integração entre **redes TA e TI** para extração de dados de banco SQL industrial
- Histórico de alarmes por tipo, horário de acionamento e duração até retirada
- ETL via **Power Query em Dataflows** na nuvem
- Indicadores: frequência por área/tipo, criticidade, parâmetros de segurança de processos

**Stack:** `Power BI` `SQL` `Power Query` `Dataflows` `Redes TA/TI`

---

### 🗄️ Gestão de Data Warehouses — 2 Plantas Multinacionais
> End-to-end de dois DWs industriais em ambiente multinacional

- Ingestão de fontes heterogêneas: **SQL Server, Oracle, APIs REST, OLEDB, ODBC**
- Pré-ETL customizado por caso de uso
- Governança de dataflows e pipelines Python
- Responsável por toda a cadeia: da extração na fonte até a entrega ao usuário final

**Stack:** `SQL Server` `Oracle` `OLEDB/ODBC` `Power BI Dataflows` `Python` `APIs REST`

---

### 🔁 Reestruturação PI System & Digital Twin da Planta
> Infraestrutura PI redundante e espelho digital do supervisório

- Reestruturação das VMs: **PI Vision, PI AF, SQL Server, PI Data Archive**
- Comunicação redundante entre interfaces **OPC DA/UA** com failover automático
- Desenvolvimento de *digital twin* da planta em PI Vision para engenharia, produção e gestão
- Análises de falhas de equipamentos e correlação de desvios de produção via **PI AF Analytics**

**Stack:** `PI Vision` `PI AF` `OPC DA/UA` `Windows Server` `PI Data Archive`
