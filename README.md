## 🚀 Projetos em Destaque

### 🏭 Batch Tracking — Rastreamento de Lote Industrial
> Algoritmo de rastreamento retroativo de lotes ao longo de ~15 áreas do processo fabril

- Lógica de varredura reversa utilizando **PI AF Event Frames** como flags de passagem de produto
- Consolidação de parâmetros de processo e qualidade por etapa
- Report **Power BI** integrado com API JSON do sistema de rastreabilidade de matéria-prima
- Cruzamento de notas fiscais, volumes de entrada e histórico completo de fabricação

**Stack:** `PI AF` `Event Frames` `PI Tags` `Power BI` `API JSON` `Python`

**Stack:** `Power BI` `SQL` `Power Query` `Dataflows` `Redes TA/TI`
