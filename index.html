Projetos Selecionados

PI System · Event Frames · Power BI

Batch Tracking — Rastreamento de Lote Industrial (Planta Limeira)

Algoritmo de rastreamento retroativo de lotes de produto ao longo de ~15 áreas do processo fabril. A partir do número de lote inserido no estágio final, o sistema percorre cada etapa anterior identificando os flags de passagem de volume, os parâmetros de processo e os dados de qualidade registrados nos Event Frames do PI AF. O resultado é consolidado em um report Power BI que integra todos os Event Frames com uma API JSON do sistema de rastreabilidade de matéria-prima — cruzando notas fiscais, volumes de entrada e o histórico completo de fabricação do produto final.

Plataforma de Alarmes Industriais — Monitoramento e Segurança de Processos

Solução end-to-end de monitoramento de alarmes para plantas industriais, desde a integração entre redes TA e TI até a entrega de indicadores estratégicos em Power BI. Reestruturação completa das VMs da infraestrutura PI System (PI Vision, PI AF, SQL Server, PI Data Archive/Historiador), implementando comunicação redundante entre interfaces OPC — com failover automático onde cada interface opera como backup da outra.
